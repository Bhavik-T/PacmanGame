Description of Project:
    Pacman 112 is a parody of the original Pacman game. 
    The game consists of a player that attempts to collect all the points before the ghost kills them.
    You can also create your own custom map to play your game in!

How to run the project:
    To run the project, open the file: "PacManGame.py" and hit run (or command + b)!

Which libraries you're using that need to be installed, if any:
    This game doesn't use any additional libraries and therefore, nothing needs to be installed

A list of any shortcut commands that exist:
    You can end the game by pressing "p." This will take you to the win game screen where you can save scores and view leaderboard.
